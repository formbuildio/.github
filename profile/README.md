## 🚀 Official Tools & SDKs

We maintain a collection of open-source SDKs and templates to help you integrate `formbuild.io` into your projects in seconds.

### Libraries
- [**@formbuild/react**](https://github.com/formbuildio/react) - The official React hooks library (`useFormbuild`). Zero dependencies, fully typed.

### Starter Templates
- [**Next.js Tailwind Starter**](https://github.com/formbuildio/nextjs-starter) - A beautiful, ready-to-deploy Next.js 14 App Router template featuring a fully wired-up contact form. 
- [**Vanilla HTML Starter**](https://github.com/formbuildio/html-starter) *(Coming soon)* - The absolute simplest way to collect submissions without JavaScript.

## 🤖 Built for AI Agents
`formbuild.io` is designed from the ground up to be the missing backend for AI code generators like **v0.dev**, **Cursor**, and **Bolt**. 
When you prompt an AI to build a UI with a contact form, simply give it your unique `formbuild.io` endpoint, and your form is instantly alive.

## 🔒 Security
If you believe you have found a security vulnerability in any of our open-source tools or our core platform, please read our [Security Policy](SECURITY.md) and report it directly to `security@formbuild.io`.
