## Open Source Ecosystem

We maintain several open-source libraries and templates to simplify integration with Formbuild. 

### SDKs
- [**@formbuild/react**](https://github.com/formbuildio/react) - The official React package. Provides the `useFormbuild` hook for managing submission state, data parsing, and error handling.

### Boilerplates
- [**Next.js Starter**](https://github.com/formbuildio/nextjs-starter) - A minimal Next.js 14 template (App Router) integrated with Tailwind CSS and `@formbuild/react`. Use this to quickly bootstrap a new project.

## AI Integration

Formbuild is designed to work seamlessly with AI code generators like Cursor and v0.dev. By providing your endpoint to the AI, it can scaffold functioning forms that connect directly to your database without any additional backend configuration.
