# Security Policy

## Supported Versions

Currently, the latest versions of our SDKs are fully supported with security updates.

| Project | Supported Versions |
| ------- | ------------------ |
| @formbuild/react | > 0.1.0 |
| formbuild.io API | v1 |

## Reporting a Vulnerability

We take the security of our platform and open-source tools very seriously. 

If you have discovered a vulnerability, please do **not** open a public issue on GitHub. Instead, email us directly at:

**security@formbuild.io**

Please include the following information in your report:
- Type of issue (e.g. buffer overflow, SQL injection, cross-site scripting)
- Full paths of source file(s) related to the manifestation of the issue
- The location of the affected source code
- Any special configuration required to reproduce the issue
- Step-by-step instructions to reproduce the issue
- Proof-of-concept or exploit code (if possible)
- Impact of the issue

We will send an initial response to your report within 48 hours and keep you updated as we investigate and resolve the issue.
